# R-TYPE TACTICS ダメージシミュレータ

[English README is available here](README.en.md)

R-TYPE TACTICSの各種武器の攻撃・迎撃・反撃のダメージや回避率をシミュレートします。いろいろなデータを解析して作っています。

アクセスはこちら:

- https://106-.github.io/r-type-tactics-damage-simulator/
- https://r-11s2.vercel.app/

ヘッダーの「解析知見」から、復元した戦闘フロー、回避式、属性相性表、地形・迎撃・熟練・特殊攻撃の要点を参照できます。同じ内容を下記の[解析知見](#解析知見)にも掲載しています。

## 現在の出力

- 実効回避（地形ブロック）率と命中率、必中バイパス設定に基づく必中表示
- 命中時ダメージの最小・RNG平均・最大
- 回避を含む期待ダメージ
- 対象最大HPに対する命中時・回避込みの損耗率
- 5機編隊を満タンから攻撃した場合の損耗機数
- 属性相性、実効地形防御、迎撃による被ダメ減衰率
- 攻撃属性と対象の所持武器に連動した迎撃武器選択（攻撃・命中値・熟練・双方の現在HP率を反映）
- 通常/反撃、ユニット固有の編隊数（1/5）と損耗比、ノックバック衝突、半減フラグ
- 対象のユニット種別と占有HEX数は選択ユニットから自動設定
- 攻撃側・対象側とも、熟練補正先を表示し、HP・回避率・武器AP・命中率へ自動適用
- ユニット名検索、陣営（人類・バイド・その他）、種別（艦船・編隊・フォース）で絞り込めるユニット選択画面

## 解析知見

### 01 戦闘フロー

1. **攻撃成立** — 攻撃武器・通常/反撃・射程などを決定
2. **迎撃** — 対象攻撃だけ武器対武器の減衰率を算出
3. **地形ブロック** — 実効回避率に対する乱数1回で命中判定
4. **HPダメージ** — 武器威力、編隊、迎撃、威力乱数、地形防御、属性相性を適用
5. **追加処理** — ノックバック衝突、編隊機数、演出用の表示値を更新

```
ダメージ ≈ 武器威力 × 編隊率 × (1−迎撃率) × 威力乱数 × (1−実効地形防御) × 属性相性
```

ダメージの威力乱数は、下記の各範囲内から一様に選ばれます。通常攻撃ではダメージが下振れ、一方反撃ではダメージが上振れるように乗ります。

| 威力乱数 | 範囲 | 平均 |
| --- | --- | --- |
| 通常攻撃RNG | 57.5～100% | 78.75% |
| 反撃RNG | 100～142.5% | 121.25% |

### 02 回避・命中

```
実効回避 = 対象回避 + [回避専念時: 対象回避 × 0.5 / 占有HEX数] + 地形回避 − 武器命中
```

0～100%に制限後、24bit一様乱数を1回比較します。「回避に専念」では、基礎回避率の `0.5 / 占有HEX数` 倍を加算します。反撃・迎撃時はこのボーナスを受けません。シミュレーターでは回避専念と迎撃武器を排他選択します。

> **ゲーム内UIの表示注意:** 「回避・防御に専念する」の「回避率 +25%」は実際の加算値でも基礎回避率でもありません。専念ボタンの数値欄が更新されず、Widgetデザイナの初期値 `+25%` が残る表示バグです。実際の専念ボーナスは基礎回避率と占有HEX数から別途計算されます。また「防御」に相当する被ダメージ軽減は確認できません。

複数HEXユニットの地形回避は、占有HEX全体ではなく**ユニットの基準HEXにある地形だけ**を参照します。端のHEXに地形補正があっても加算されず、基準HEXに補正があれば占有HEX数で割らず全量が1回加算されます。

#### 必中バイパス

```
必中バイパス設定あり → 回避率 0% / 命中率 100%
```

この設定を持つ攻撃は、対象回避・地形回避・武器命中による通常の回避計算を通りません。表示命中値100%や粒子属性そのものが直接の条件ではありません。

- 収録中の**チャージ武器131定義はすべて必中バイパス設定済み**です。チャージ武器以外の一部の特殊攻撃も含まれます。
- **表示命中値が100%未満でも必中です。** アイビーロッドは表示75%ですが、必中バイパス対象のため実戦では回避計算を通りません。

### 03 武器属性とユニット相性

武器には、ダメージ相性を決める属性と、直行・偏光・追尾など射線や使用条件を決める分類があります。相性ダメージに使われるのは光学・機械・生体などの属性です。

| 対象系統 | 光学 | 機械 | 生体 | 粒子 | 火炎 | 精神 | 氷 | 酸 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 機械 | ±0 | ±0 | +20% | ±0 | ±0 | ±0 | +10% | +10% |
| 生物 | +10% | +20% | ±0 | +10% | +25% | +15% | +15% | +15% |
| 岩石 | ±0 | ±0 | ±0 | ±0 | ±0 | ±0 | ±0 | ±0 |
| 氷 | +10% | ±0 | −10% | ±0 | +30% | ±0 | ±0 | ±0 |
| 異質 | ±0 | ±0 | ±0 | ±0 | ±0 | ±0 | ±0 | ±0 |

"異質"はグリッドロックや琥珀色の瞳孔などに振られています。

### 04 地形回避・地形防御・部分遮蔽

- **地形回避**は命中判定のブロック率に固定加算されます。
- **地形防御**は命中後のダメージ減衰です。現在確認できた実HP式では光学属性に適用され、機械属性ミサイルなどはバイパスします。
- **機械壁・生体壁**は地形回避と光学地形防御の特例対象です。
- **部分遮蔽**は最大射程2で中間経路の片方だけが塞がる状態。機械属性以外のダメージを50%にします。

### 05 迎撃は武器対武器の比例減衰

機械・生体・氷属性の攻撃に対して、対象が持つ迎撃武器が候補になります。

攻撃距離を個別に選ぶのではなく、攻撃武器と迎撃武器の射程が1HEXでも重なれば、その共通射程で迎撃可能として計算します。射程が一部だけ重なる場合は、候補欄に迎撃可能な距離を表示します。

「迎撃距離制限の緩和」をONにすると、ユニットの形状によって迎撃できる状況として、射程の重なりだけを無視します。属性や迎撃可否など、ほかの条件は維持されます。

```
未補正迎撃率 = (迎撃武器威力 / 攻撃武器威力) × (迎撃側現在HP / 最大HP) × (攻撃側最大HP / 現在HP) × 迎撃武器命中
```

| 未補正迎撃率 | 適用値 |
| --- | --- |
| 10%未満 | 10%（最低迎撃率） |
| 10～90%未満 | 計算値のまま減衰 |
| 90%以上 | 100%（完全迎撃） |

鹵獲弾は例外的に迎撃率0%です。迎撃は地形防御より前に処理されます。

#### 体当たりに対する迎撃

体当たり・フォースシュートなどはノックバックを発生させます。移動先が地形またはユニットで塞がると追加基礎ダメージ **+25** です。

体当たり・フォースシュートを迎撃候補となった武器で迎撃すると、対象側ダメージの減衰に加えて攻撃側に反動ダメージが入ります。

```
反動 = 攻撃側現在HP × 迎撃率 × 0.85（最低25・最大115）
```

| 武器定義上の弾数 | 係数 |
| --- | --- |
| 2以上 | ×0.85（現行の全迎撃武器） |
| 1 | ×1.05（コード上の分岐のみ。現行データでは該当なし） |

この分岐が見るのはゲーム中の残弾数ではなく、武器定義上の初期・最大弾数です。現在残弾は別に管理されるため、残り1発になっても係数は変わりません。抽出した迎撃武器99定義はすべて定義弾数2以上であるため、シミュレーターでは係数を **0.85固定** としています。

反動は最低25、最大115。この反動で攻撃側が撃破される場合、対象側への体当たりダメージは0になります。迎撃しない場合や、迎撃候補にならない場合は発生しません。

### 06 鹵獲弾の対象条件と成功率

鹵獲弾は通常の武器命中・対象回避計算とは別の専用判定を使います。武器データ上の命中値92%は鹵獲成功率ではありません。

```
鹵獲可能条件 = 残HPが0%超～25%以下 または 残燃料が0～40%以下
```

| 状況 | 鹵獲成功率 |
| --- | --- |
| 通常時 | 60% |
| 回避扱い | 15% |
| HP・燃料条件外 | 0%（鹵獲不可） |

- **HP条件**は最大HPの75%以上を削り、残HPを25%以下にすると成立します。25%ちょうどを含み、HP 0は含みません。
- **燃料条件**は残燃料40%以下で成立します。HPが高くても、燃料を60%以上消費させれば候補になります。
- **HPをさらに減らしても成功率は上がりません。** 残HP25%でも1%でも、通常60%・回避扱い15%です。
- **対象外**は旗艦、大型、親子・接続中のユニットなどです。攻撃失敗扱いの場合も鹵獲できません。
- **鹵獲弾は迎撃率0%**の特例で、成功後はミッション中だけ自軍ユニットになり、終了後に資源へ解体されます。

### 07 チャージ・特殊武器

- **波動砲などのチャージ武器**は、チャージ・範囲選択・専用演出に特殊分岐がありますが、命中後のHPダメージは基本的に表記APから同じ系統へ合流します。
- 命中判定では、収録中のチャージ武器はすべて上記の**必中バイパス**が適用され、対象や地形の回避率を無視します。
